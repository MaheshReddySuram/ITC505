Revenge of Lights Out

Addendum

This implementation of Lights Out was created with the following considerations:

  • The game board is a 5x5 grid created dynamically using JavaScript.
  • Each click toggles the clicked cell and its adjacent cells (top, bottom, left, right).
  • The initial board state is generated by simulating random clicks to ensure it's solvable.
  • Win condition is checked after every move by verifying all cells are white.
  • The design uses a simple black and white color scheme with Comic Sans MS font for a playful look.

Challenges & Solutions:

  • Initially struggled with generating random but solvable boards. Solved by starting with all lights on and simulating random valid moves.
  • Ensuring boundary conditions worked correctly (edge/corner cells). Solved by carefully checking neighbor indices in getNeighbors().
  • Implementing smooth transitions between states. Used CSS transitions for visual feedback.
